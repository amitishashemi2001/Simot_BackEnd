# SIMOT Trade - Backend API

## Overview

This is the backend API for **Silk Road Modern Trade (SIMOT Trade)** , a commercial company specializing in import and export services. The project is currently 
under active development.

## Technologies

- **Python** 3.8+
- **Django** 4.x
- **Django REST Framework** 3.14+
- **SQLite** (development environment)
- **PostgreSQL** (planned for production)
- **Docker** (coming soon)

## Current Status

- ✅ Django REST Framework backend implemented
- ✅ Core app structure (home, about, contact, news, services, education, gallery)
- ✅ REST API endpoints configured
- ⏳ Dockerization (in progress)
- ⏳ PostgreSQL migration (planned)
